# NITA bot

NITA botは、マリオカート8DXにおけるアイテム使用なしのタイムアタック (No Item Time Attack) の記録を登録および閲覧する機能を持つbotです。

## コマンドリスト
```
<必須> [任意]

記録の登録
_s <track> <time>

記録の表示（コース, sub time, ユーザ名のいずれかを指定可能）
_r [track] [subtime] [username]

チームの記録の表示
_t <track>

WRの表示
_wr <track>

tier別平均タイムの表示
_tt <track>

記録の削除
_d <track>

解説動画の送信
_v <track>
```
